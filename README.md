# ittaspajtatanc.hu

A very simple project to get my feet wet with Docker and GitHub Workflows.

It leverages:
- [nginx](https://github.com/nginx/nginx) as the http server for the static files in the `public` folder.
- [Departure Mono](https://github.com/rektdeckard/departure-mono) font for the ASCII art.
- GitHub Workflows to build the container and push it to GHCR.
